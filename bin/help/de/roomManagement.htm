Zimmer-Verwaltung
Die Zimmer-Verwaltung kann Neue Zimmer anlegen und vorhandene editieren oder löschen.
Es können mehrere Aktionen gleichzeitig ausgeführt werden (z.B. neues Zimmer anlegen, vorhandenes Zimmer editieren und ein vorhandenes zum Löschen vormerken). Die Änderungen werden aber erst nach dem Speichern wirksam.

Vorhandene Zimmer werden im Belegungs-Plan angezeigt und können zur Erstellung neuer Belegungen verwendet werden.
Die Farbe eines Zimmers wird zur Markierung einer Belegung im Belegungs-Plan verwendet.

Neue Zimmer erstellen
Neue Zimmer werden über das Plus oben rechts in der Tabelle hinzugefügt, werden der Tabelle angefügt und grün hinterlegt.
Damit neue Zimmer abgespeichert werden können, müssen alle Felder gefüllt sein.
  • Bezeichnung des Zimmers
  • Art (z.B. DZ für Doppelzimmer oder EZ für Einzelzimmer)
  • Preis pro Nacht im Format "00,00" (z.B. 50,00)
  • Farbe wird automatisch beim Speichern gesetzt (kann später verändert werden)
Neue Zimmer werden erst abgespeichert, wenn der Button "Speichern" gedrückt wird und sind anschließend nicht mehr grün hinterlegt. Die Erstellung kann durch das "X" am Ende der Zeile abgebrochen werden. Die Zeile wird wieder entfernt.

Vorhandenes Zimmer bearbeiten
Die Werte eines Zimmers können direkt in der Tabelle editiert werden. Die Farbe eines Zimmers kann über einen Klick auf das Farbfeld geändert werden.
Änderungen werden erst durch einen Klick auf den Button "Speichern" wirksam.

Vorhandenes Zimmer löschen
Ein Zimmer kann durch das Klicken auf das "X" am Ende der Zeile des jeweiligen Zimmers zum Löschen markiert werden. Das / die Zimmer werden erst gelöscht, wenn der Button "Speichern" geklickt wird.
